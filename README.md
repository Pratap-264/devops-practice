# DevOps Practice Project

## Overview
This project generates automated system logs using Python.

## Features
- Automated logging
- Timestamp-based logs
- Linux file handling
- Basic system monitoring

## Technologies Used
- Python
- Linux
- Git & GitHub

## Run the Project

```bash
python3 system_check.py
```
